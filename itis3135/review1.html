Page Structure Checklist

  • Design Requirements:
    • Sufficient contrast and font sizing ☑️
    • Site colors and fonts using standard CSS ☑️
    • CRAP principles (Contrast, Repetition, Alignment, Proximity) ☑️
  • Page Structure:
    • Header, Main, Footer elements ☑️
    • Optional: nav between header and main ☑️
  • Header Requirements:
    • Site/brand header with H1 ☑️
    • Branded pages: company/brand name ☑️
    • DO NOT include page name in header ☑️
  • Main Requirements:
    • Page name as H2 (not site/brand name) ☑️
    • Unique page content ☑️
  • Footer Requirements:
    • Navigation menu in nav element ✖️
    • HTML validation button/link ☑️
    • CSS validation button/link ☑️
  • Assignment Requirements:
    • Layout including header, footer, main, id and class tags ☑️
    • Consistency of layout across pages ☑️
    • appropriate content on the home page including image/s ☑️
    • Link to all 4 mandatory pages and links work on all the pages; bar is consistent position across pages ☑️
    • Appropriate content on each page including image/s ☑️
    • Website is aesthetically pleasing and easy to navigate ☑️
  • Notes:
    • Overall the website looks great! A small tip could be to reduce the size of the images so it fits on 16:9 screen sizes and the user can see it all without having to scroll.
